Welcome to StationeersMods

Installation Instructions

StationeersMods is a Bepinex plugin that helps you load mods you subscribed to from the workshop. Follow the instructions below to get started.

  1. If you haven't installed Bepinex yet, click here to install it.
  2. Download the latest StationeersMods.zip from Releases and extract it to the BepInEx/plugins/ folder.
  3. Install mods from the workshop and enjoy your game!

For help, join us on Discord.
